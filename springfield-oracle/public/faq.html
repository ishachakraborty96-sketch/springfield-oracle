Springfield Oracle is the definitive open-source tracker for Simpsons predictions. Every prediction is logged with verified episode references, real-world event citations, and honest fact-checks. No clickbait, no AI-generated fakes — just sourced data.
No. Springfield Oracle is an independent, community-powered project. It is not affiliated with Fox, Disney, The Simpsons, or any official entity.
Each prediction requires: a verifiable season number and episode code, an original air date, at least one external source for the real-world event, and an honest status assessment. Predictions without a verified episode source are not accepted.
Confirmed = The prediction has occurred. Sourced and verified.

Pending = Has not yet occurred, or is currently unfolding.

Disputed = The connection is contested, or the viral version misrepresents the source.
You can submit predictions directly on the homepage using the submission form, or open a Pull Request on GitHub adding to the predictions database. Every submission is reviewed and sourced before it goes live.
Many viral clips circulating on TikTok and X/Twitter claiming "The Simpsons predicted [event]" are AI-generated deepfakes or taken out of context. Springfield Oracle verifies every claim against the actual episode. If a prediction is misleading, we flag it as Disputed.
The viral score (1-10) represents how widely discussed a prediction is on social media. It does not indicate accuracy — only cultural reach and engagement.
